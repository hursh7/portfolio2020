************ 내 branch 에서 master에 올릴 때 **************
내 branch : 수정 -> 커밋 -> 푸쉬 -> master로 이동 -> pull -> (터미널 - 새 터미널) 에서 git merge jun -> push

************ master에서 내 branch로 가져올 때 **************
master : pull -> 내 branch로 이동 -> (터미널 - 새 터미널) 에서 git merge master -> push

////////////////// 주의사항 //////////////////////
  작업할 때는 반드시 내 branch에서 작업할 것!!!!!! 
   master에서 항상 pull 받는 습관!!
